# 文章模板

## 专题文章模板

适用于围绕单一话题深度整理多条相关资讯的场景。

### Frontmatter

```yaml
---
title: "{话题}最新动态：{核心发现}"
subtitle: "{1-2句话概括}"
author: "资讯助手"
date: "2026年03月17日"
---
```

### 正文结构

```markdown
开篇导语（1-2 段，100-200 字）：
- 概括本文覆盖的话题范围
- 点明最重要的 2-3 个发现

## 资讯条目 1 标题

背景介绍（1-2 句）。

核心内容（2-3 段）：
- 发生了什么
- 为什么重要
- 关键数据或引用

> 关键引用或数据

相关链接：[来源名称](URL)

## 资讯条目 2 标题

... 同上结构 ...

## 资讯条目 N 标题

... 同上结构 ...

---

文末总结（可选，1 段）：
- 整体趋势判断
- 值得关注的后续发展
```

### 写作原则

| 原则 | 说明 |
|------|------|
| 事实优先 | 基于搜索到的真实信息，不编造 |
| 引用来源 | 每条资讯附带原始链接 |
| 简洁准确 | 每个分区 200-400 字 |
| 代码展示 | 技术类内容附带关键代码片段 |
| 数据引用 | 用 blockquote 突出关键数据 |

### 标题命名参考

| 话题类型 | 标题模板 |
|---------|---------|
| 技术进展 | `{技术} 最新突破：{核心发现}` |
| 产品发布 | `{公司} 发布 {产品}：{亮点}` |
| 行业趋势 | `{行业} {时间段}趋势洞察` |
| 深度分析 | `深度解读：{话题} 的现状与未来` |
